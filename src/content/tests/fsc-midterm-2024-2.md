## Exam Information

- **Duration**: 2 hours
- **Allowed materials**: RISC-V reference card (provided)
- **Total points**: 100

## Topics Covered

- Number representation (binary, hexadecimal, two's complement)
- RISC-V instruction formats and encoding
- Assembly programming (loops, conditionals, procedures)
- Stack usage and calling conventions
- Data representation in memory

## Question Summary

1. **(20 pts)** Number conversions and two's complement arithmetic.
2. **(25 pts)** Encode/decode RISC-V instructions.
3. **(25 pts)** Write a RISC-V assembly function that performs binary search on a sorted array.
4. **(15 pts)** Analyze a given assembly program and determine its output.
5. **(15 pts)** Explain the RISC-V calling convention and draw the stack frame for a recursive function.
